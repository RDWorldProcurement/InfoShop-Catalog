## Executive Summary

OMNISupply.io is an AI-powered unified procurement platform designed for enterprise customers of Infosys Limited. The platform combines advanced AI capabilities with traditional procurement workflows to deliver intelligent sourcing, quotation analysis, and supplier management.

### Core Value Proposition
- **30M+ Industrial MRO Products** - Comprehensive catalog coverage
- **100K+ Professional Services** - IT, consulting, maintenance services
- **3 AI Engines** - GPT-5.2, Claude Sonnet 4.5, Gemini 3 Flash for multi-perspective analysis
- **Real Document Extraction** - AI-powered parsing of PDF, DOCX, XLSX quotations
- **Automated UNSPSC Classification** - Deep semantic product categorization

## Appendix B: API Endpoints Summary

### Authentication
- `POST /api/auth/login` - User login
- `POST /api/auth/register` - User registration

### AI Agent
- `POST /api/ai-agent/conversation` - Main conversational endpoint

### Catalog
- `GET /api/catalog/products` - Product search
- `GET /api/catalog/services` - Service search
- `GET /api/catalog/categories` - Category list

### Quotations
- `POST /api/procurement/quotation/upload` - Demo upload
- `POST /api/procurement/quotation/upload-with-ai` - AI analysis
- `GET /api/procurement/quotation/{id}` - Get quotation

### Negotiation
- `GET /api/negotiation/strategies` - Get playbooks
- `POST /api/negotiation/generate-targets` - Generate targets
- `POST /api/negotiation/generate-email` - Draft email

### Buying Desk
- `POST /api/procurement/buying-desk/engage` - Engage with context
- `GET /api/procurement/buying-desk/requests` - User requests

### Cart & Orders
- `POST /api/cart/add` - Add to cart
- `GET /api/cart` - Get cart
- `POST /api/orders` - Create order
